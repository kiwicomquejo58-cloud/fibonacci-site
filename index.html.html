Sequência de Fibonacci

A sequência de Fibonacci é uma sucessão por recorrência, em que cada termo resulta da soma dos dois anteriores.

Exemplo: 1, 1, 2, 3, 5, 8, 13...

Para que serve o programa

Este programa serve para gerar automaticamente a sequência de Fibonacci a partir do número de termos escolhido.

Assim, conseguimos visualizar rapidamente os valores da sequência e perceber melhor como funciona esta sucessão por recorrência.

Programa

Explicação do código

n = int(input("Quantos termos? "))

Pede ao utilizador o número de termos e converte esse valor para inteiro.

a = 1 b = 1

Define os dois primeiros termos da sequência.

print(a) print(b)

Mostra os dois primeiros valores da sequência.

for i in range(n - 2):

Cria um ciclo que repete o processo para calcular os restantes termos.

c = a + b

Calcula o próximo termo somando os dois anteriores.

print(c)

Mostra o novo valor calculado.

a = b b = c

Atualiza os valores para continuar o processo nas repetições seguintes.

Conclusão

Este programa permite compreender de forma simples como funciona a sequência de Fibonacci e como a programação pode ajudar a representar conceitos matemáticos.
