Sequência de Fibonacci

A sequência de Fibonacci é uma sucessão por recorrência, em que cada termo resulta da soma dos dois anteriores.

Exemplo: 1, 1, 2, 3, 5, 8, 13...

Para que serve o programa

Este programa gera automaticamente os termos da sequência de Fibonacci, mostra o termo escolhido e calcula também as razões entre termos consecutivos.

Assim, torna-se mais fácil perceber a ligação entre a sequência de Fibonacci e a aproximação ao número de ouro.

Programa

Explicação do código da sequência de Fibonacci

n = int(input("Quantos termos? "))

Pede ao utilizador um número e transforma esse valor num inteiro.

a = 1 b = 1

Define os dois primeiros termos da sequência.

print(a) print(b)

Mostra os primeiros valores da sequência.

for i in range(n - 2):

Cria um ciclo que repete o processo para calcular os restantes termos.

c = a + b

Calcula o próximo termo somando os dois anteriores.

print(c)

Mostra o novo valor.

a = b b = c

Atualiza os valores para continuar o processo.

Número de ouro

O número de ouro é um valor aproximado de 1,618.

Quando dividimos um termo da sequência de Fibonacci pelo termo anterior, os resultados aproximam-se cada vez mais desse valor.

Exemplo: 3/2 = 1,5, 5/3 ≈ 1,667, 8/5 = 1,6. À medida que avançamos na sequência, as razões ficam mais próximas de 1,618.

Conclusão

Este programa ajuda a compreender a sequência de Fibonacci e mostra também como as razões entre termos consecutivos se aproximam do número de ouro.
